### LikedBy – Chrome Extension (YouTube)

LikedBy is a Chrome extension that enables users to track liked YouTube videos and explore liked content within their trusted circle. It enhances content discovery through social-based filtering.

### Dje – Self-Hosted Discord Music Bot

Dje is a Python-based Discord music bot designed for self-hosted environments. It provides full control over playback, supports YouTube search/URL playback, playlist handling, Spotify link resolution (via YouTube), localized UI, and persistent per-guild configurations.

Repository: https://github.com/gkhntpbs/Dje
